# Yoshi-Rules-Super-Mario-World-SMW-Hack
Here is my first SMW Hack: "Yoshi Rules SMW Patch"!  I used Powerup Patch 3.4 [by Taxiham, Locb Inc, Lx5] as a base and then added my own stuff and fixed some original bugs. You will have yoshis now in almost every single level, you'll have a power up station in each zone (like Top Secret Area but way better) where you can load up on your favorite power ups, yoshis and get moon lives, and I made a new level where the game starts off playing my own custom "insane level". I also added additional power ups throughout the game. Hope you enjoy!!
